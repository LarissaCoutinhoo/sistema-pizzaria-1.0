# Sistema de Pizzaria (Console - Java) 🍕

## Sobre o Projeto

Este é um sistema de gestão de pedidos para pizzarias desenvolvido em **Java**, executado via terminal.

---

## Objetivo do Projeto

Este projeto faz parte da minha evolução nos estudos de programação.  
Foi desenvolvido após concluir um curso de **Algoritmos e Lógica de Programação**, como primeiro sistema funcional completo.

Futuramente, o sistema poderá evoluir para:

- Versão utilizando Programação Orientada a Objetos (POO)
- Integração com banco de dados
- Interface gráfica
- API REST

---

## Funcionalidades

- Registrar novo pedido
- Capturar nome do cliente
- Registrar horário do pedido automaticamente
- Selecionar múltiplos itens do cardápio
- Calcular total do pedido
- Listar todos os pedidos realizados no dia
- Mostrar faturamento total do dia
- Menu interativo em loop

---

## Tecnologias Utilizadas

- Java
- Scanner 
- ArrayList
- LocalDateTime

---
## Como Executar

1. Clone o repositório
2. Abra o projeto em sua IDE (IntelliJ, Eclipse ou VS Code)
3. Compile e execute a classe principal
4. Utilize o menu interativo no terminal
